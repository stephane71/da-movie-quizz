Da Movie Quizz
https://gist.github.com/nicolaschenet/9763332fbe09a2b1a39a

--- Général
Chaque round: acteur + affiche film
	if ok => next round
	else if nok => game over

--- Etats
Etat initial
	jeu arrêté
	boutton play => le joueur click on lance le jeu

Jeu en cours
	timer
	nombre de bonnes réponses
	acteur + affiche de film
	bouttons OUI/NON
	Si bonne réponse
		next round
	Si mauvaise réponse
		Game over
	voir les hints 1&2

Game over
	time stoped
	affichage => temps écoulé + Nombre de bonne réponses
	Si nouveau record
		proposer  inscription dans les Highscores
	boutton Play Again

Highscores
	10 meilleures parties classées par nombre de bonnes réponses

------------------------------------------------------------------

Les routes
	/play
		Etat inital => {{partial}} template
		Jeu en cours => chargement du modèle avec requêtes API (2 films pour commencer)
	/over
	/highscores
